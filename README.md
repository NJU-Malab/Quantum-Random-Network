# Quantum-Random-Network
Data for “Photonic realization of a subgraph extraction in a quantum random network”

All data supporting the findings of this study are organized according to the corresponding figures in the main text and Supplementary Information. The data are provided in two formats: raw data files (.dat) and partially processed photon-counting data (.mat).

Fig. 3 – The data consist of two-fold coincidence counts measured under different basis settings in the Z-basis. The extracted coincidence counts are stored in the file CC_router.mat, and the data processing procedure follows the script draw_poss_router.m.

Fig. 4 – The raw data and the corresponding processing script are origin_data.mat and code_for_posisErr_V.m, respectively. Necessary explanations are included as comments within the script.

Fig. S1 – Raw data for the grating coupler (GC) and edge coupler (EC) characterizations are saved in .dat format. The file CH1_background.dat records the laser background noise. The files GC_long and GC_short correspond to the long-waveguide GC and short-waveguide-connected GC described in the text, respectively. The insertion loss of the EC is characterized via the BS method.

Fig. S3 – Classical transmission spectra of all light sources S1–S8 are stored in standard .dat files. The first two columns represent the input-drop and input-through transmission spectra, respectively. The folder Result contains the transmission spectra of S1–S8 under the triple-critical condition.

Fig. S4 – Raw data from the RHOM scan.

Fig. S5 – Raw measurement data of the HHOM, together with the monitoring data of the incident power fluctuation.

All data are available from the corresponding author upon reasonable request.
